**Introduction:**
Hand-on labs are being used to teach students about emerging technologies such as cloud computing, IoT gateway, edge cloud, and Fog computing. These labs provide students with hands-on experience and help them develop an intuitive understanding of these technologies [1]. The labs cover a wide range of topics, including acquiring images, creating immutable records using cryptographic hash functions, and transmitting and storing data on the cloud [2]. They also focus on the end-to-end processing involved in these technologies, such as camera-based security applications, basic blockchain technology, and cloud computing [3]. Additionally, there are labs designed to teach students about IoT applications and different deployment models, including IoT+Edge, IoT+Cloud, and Fog [4]. These labs have been found to be effective in teaching students about security principles, ideas, and technologies in information technology [5].
